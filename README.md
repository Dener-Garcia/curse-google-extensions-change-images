# Extensão Change Images

Uma extensão para google chrome que troca todas as imagens de uma página pela imagem que você deseja. 

<img src="demonstration.gif" width="800px"/>

## 🔥 Funcionalidades
- ✅ Trocar todas imagens por outra imagem a sua escolha

## ⬇️ Rodando o projeto

1. Clone esse repositório ou faça o download dos arquivos para sua máquina.
2. Abra o VS Code dentro da pasta do projeto
3. Abra suas extensões no browser
4. Habilite o modo developer
5. Click em Load unpacked
6. Selecione a pasta do projeto e pronto
7. Abra um navegador baseado em chrome e a extensão está pronta

## 🤔 Aprendizados
Criar uma extensão para o navegador é complexo no sentido de respeitar a documentação própria, a sintaxe e exigências para poder rodar a extensão deve ser bem definida, O debug é mais complicado pois o que a extensão enxerga é somente uma função que injetamos nela.
